# Motorola One Fusion Overlay

## Description
This module installs all necessary rro-overlays for the correct function of the GSI in the Motorola One Fusion (A11, A12 & A12.1 ONLY).

## Compatibility
- Android 11R
- Android 12S
- Android 12L

## Notes
- Only for GSI
